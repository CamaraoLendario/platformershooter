The game also lacks any kind of tutorial, so, while that is still a problem here is the full list of controls:

--> For controller: (highly recommended)
(Controller commands bellow assume an Xbox controller, so translate that to whatever controller you're using)

Left joystick to Move 

A to Jump or dash as a ship

X to Melee or shoot as a ship (if you melee in the air as a pilot, you will perform a dash)

B to use any held items (for now these are only weapons for the pilot)

Right Trigger to Aim

Select to pause the game

Start to Start the game

When menuing, use the Dpad on the controller. X is confirm and B is cancel
Press Start to leave the Renaming screen

--> For Keyboard:

WASD to Move 

SpaceBar to Jump or dash as a ship

H to Melee or shoot as a ship (if you melee in the air as a pilot, you will perform a dash)

J to use any held items (for now these are only weapons for the pilot)

Shift to Aim

P to pause the game

Enter to Start the game

When menuing, use WASD. H is confirm and J is cancel
Press Enter to leave the Renaming screen
